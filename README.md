# 😂 Random Joke Generator
This is a simple web application that shows random jokes using an API.
## 🚀 Features
-Get a random joke on button click
-Uses API to fetch jokes
-Simple and clean UI
-Beginner friendly Project
## ⚒️Technologies Used
-HTML
-CSS
-JavaScript
-API(fetch)
## 📌How it works
-Click on the button
-App sends a request to joke API
-You get a random joke on screen
